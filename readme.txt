Adapter Design Pattern - 
I have created Adapter design pattern where MultipleItems and SingleItem are the Interfaces, CheckItemType and CheckItemPrice are concrete classes, AuctionItemAdapter is adapter class implementing the MultipleItems interface. My main class is AuctionItemMain.

Command Design Pattern - 
I have created Command Design Pattern where Bid is the interface, AuctionItem is the request class, RaiseBid and AcceptBid is a concrete class implements Bid, Auctioneer is the broker class. The BidManagementMain is the main class. In result we can see if the item is being bid on and bids being raised.

Facade Design Pattern - 
I have created ShowInventoryMain as the main class, Item as interface, Painting, Table and Vase are concrete classes implementing the same interface and ItemFacade as facade class.

Factory Design Pattern - 
I have created ItemTypeMain as the main class, Item as interface, Painting, Table and Vase as concrete classes implementing the same interface and ItemFactory as factory class to generate object of concrete class.
